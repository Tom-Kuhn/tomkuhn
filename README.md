# tomkuhn.co.uk

Personal website for Tom Kuhn — Staff Engineer, Architect, and Problem Solver.

## Technologies

- **[Jekyll](https://jekyllrb.com/) 4.4** — static site generator
- **Ruby 3.3+** — runtime
- **Liquid** — templating language used in layouts and includes
- **Sass/SCSS** — styles at `assets/css/main.scss`, compiled by Jekyll's built-in pipeline
- **[Tailwind CSS](https://tailwindcss.com/) (Play CDN)** — utility classes; custom palette and fonts defined inline in `_includes/head.html`
- **[UIKit 3](https://getuikit.com/)** — UI component library (offcanvas nav, form components)
- **GitHub Actions** — CI/CD pipeline deploying to GitHub Pages on push to `main`

## Active Plugins

| Plugin | Purpose |
|---|---|
| [jekyll-feed](https://github.com/jekyll/jekyll-feed) | Auto-generates an Atom feed at `/feed.xml` |
| [jekyll-seo-tag](https://github.com/jekyll/jekyll-seo-tag) | Injects SEO meta tags (`<title>`, Open Graph, Twitter Card, canonical URL) via `{% seo %}` in `_includes/head.html` |
| [jekyll-archives](https://github.com/jekyll/jekyll-archives) | Auto-generates tag archive pages at `/tag/:name/` |
| [jekyll-sitemap](https://github.com/jekyll/jekyll-sitemap) | Auto-generates `sitemap.xml` |

Additional gems are present in the `Gemfile` (`jekyll-assets`, `jekyll-redirect-from`, `jekyll-tagging`, `jekyll-mentions`, `jekyll-gist`) but are not actively used by the current site.

## Prerequisites

- Ruby 3.3+
- Bundler (`gem install bundler`)

## Getting Started

Install dependencies:

```bash
bundle install
```

Serve locally with live reload:

```bash
bundle exec jekyll serve
```

The site will be available at `http://localhost:4000`.

Build for production:

```bash
bundle exec jekyll build
```

Output is written to `_site/`.

## Writing a post

Create a file in `_posts/` following the naming convention `YYYY-MM-DD-title.md`:

```yaml
---
layout: post
title: "My post title"
date: 2026-01-01
description: "A one-sentence summary used in card previews and SEO meta."
tags: [architecture, engineering, career]
---

Post content here.
```

Tag archive pages are automatically generated at `/tag/:name/` by jekyll-archives. A post only appears on the Writing page and homepage if it has at least one tag.

## Deployment

Pushing to the `main` branch triggers the GitHub Actions workflow at [.github/workflows/jekyll.yml](.github/workflows/jekyll.yml), which builds the site with `bundle exec jekyll build` and deploys it to GitHub Pages automatically.
